title: "Sessions"
description: "A session links a dashboard and an embed user. This also store parameters that can be used to alter the data source."
icon: "link"
---

### Session fields

<ParamField path="dashboard" type="string" required>
  The identifier of the dashboard associated with the session.
</ParamField>

<ParamField path="embed_user" type="string" required>
  The identifier of the embed user associated with the session.
</ParamField>

<ParamField path="created_at" type="string">
  The timestamp indicating when the session was created.
</ParamField>

<ParamField path="id" type="string">
  The unique identifier for the session.
</ParamField>

<ParamField path="parameters" type="object">
  The parameters or configuration associated with the session as a json object
  with key-pair values.
    <CodeGroup>
  ```json example
    {
        "year": 2023,
        "sort": "asc"
    }
  ```
    </CodeGroup>
</ParamField>

<ParamField path="team" type="string">
  The identifier of the team to which the session belongs.
</ParamField>
