---
title: "Dashboards"
description: "A dashboard is a collection of widgets and can be connected to multiple data sources. A dashboard is duplicated when creating a session that ties a dashboard to an embed user."
icon: "objects-column"
---

### Dashboard fields

<ParamField path="title" type="string" required>
  The title or name of the dashboard.
</ParamField>

<ParamField path="created_at" type="string">
  The timestamp indicating when the dashboard was created.
</ParamField>

<ParamField path="created_by" type="string">
  The identifier or name of the user who created the dashboard.
</ParamField>

<ParamField path="description" type="string">
  A brief description of the dashboard.
</ParamField>

<ParamField path="id" type="string">
  The unique identifier for the dashboard.
</ParamField>

<ParamField path="last_updated_at" type="string">
  The timestamp indicating when the dashboard was last updated.
</ParamField>

<ParamField path="last_updated_by" type="string">
  The identifier of the user who last updated the dashboard.
</ParamField>

<ParamField path="parent_dashboard" type="string">
  The identifier of the parent dashboard if this dashboard is a part of a
  hierarchy.
</ParamField>

<ParamField path="settings" type="object">
  The settings/configuration associated with the dashboard.

  <Expandable title="properties">
  <ParamField path="theme" type='"dark" | "light" | "auto"'>
  The theme of the dashboard, which can be "dark," "light," or "auto."
</ParamField>

<ParamField path="dark_background" type="string">
  The background color for the dark theme.
</ParamField>

<ParamField path="dark_foreground" type="string">
  The foreground color for the dark theme.
</ParamField>

<ParamField path="light_background" type="string">
  The background color for the light theme.
</ParamField>

<ParamField path="light_foreground" type="string">
  The foreground color for the light theme.
</ParamField>

<ParamField path="border_radius" type="number">
  The border radius applied to dashboard elements.
</ParamField>

<ParamField path="font" type="string">
  The font used for text in the dashboard. It can be one of `inter`,
  `helvetica`, `arial`, `verdana`, `tahoma`, `trebuchet`, `gill sans`, `times
  new roman`, `georgia`, `palatino`, `baskerville`, `andale mono`, `courier`,
  `lucida console`, `monaco` or `comic sans ms`.
</ParamField>

<ParamField path="editable" type="boolean">
  Indicates whether the dashboard is editable when embedded.
</ParamField>

<ParamField path="can_ask_questions" type="boolean">
  Indicates whether users can ask questions or interact with the dashboard when
  embedded.
</ParamField>

<ParamField path="hide_header" type="boolean">
  Indicates whether the dashboard header should be hidden when embedded.
</ParamField>

  </Expandable>
</ParamField>

<ParamField path="team" type="string">
  The identifier or name of the team to which the dashboard belongs.
</ParamField>

<ParamField path="thumbnail" type="string">
  The URL to the thumbnail image representing the dashboard.
</ParamField>
