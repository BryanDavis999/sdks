---
title: "Automations"
description: "An automation allows you to send dashboards to your users on a
  fixed schedule."
icon: "bolt"
---

### Automation fields

<ParamField path="title" type="string" required>
  The title or name of the automation.
</ParamField>

<ParamField path="created_at" type="string">
  The timestamp indicating when the automation was created.
</ParamField>

<ParamField path="created_by" type="string">
  The identifier of the user who created the automation.
</ParamField>

<ParamField path="dashboard" type="string" required>
  The identifier of the dashboard associated with the automation.
</ParamField>

<ParamField path="description" type="string">
  A description of the automation.
</ParamField>

<ParamField path="email_format" type="string">
  The format or template for the email generated by the automation.
</ParamField>

<ParamField path="email_subject" type="string">
  The subject line of the email generated by the automation.
</ParamField>

<ParamField path="enabled" type="boolean" required>
  Indicates whether the automation is currently enabled.
</ParamField>

<ParamField path="id" type="string">
  The unique identifier for the automation.
</ParamField>

<ParamField path="last_run_at" type="string">
  The timestamp indicating when the automation was last run.
</ParamField>

<ParamField path="last_updated_at" type="string">
  The timestamp indicating when the automation was last updated.
</ParamField>

<ParamField path="last_updated_by" type="string">
  The identifier of the user who last updated the automation.
</ParamField>

<ParamField path="next_run_at" type="string">
  The timestamp indicating when the automation is scheduled to run next.
</ParamField>

<ParamField path="output_format" type="string" required>
  The format or type of output generated by the automation. can be of type
  `excel`, `csv`, `link`, `pdf` or `image`.
</ParamField>

<ParamField path="recipient_type" type="string" required>
  The type or category of recipients for the automation. Can be of type
  `internal` or `external`.
</ParamField>

<ParamField path="recipients" type="string[]" required default="[]">
  An array of emails representing the recipients of the automation.
</ParamField>

<ParamField path="schedule" type="string" required>
  The schedule or frequency at which the automation runs as a cron tab. Eg: `30
  4 1,15 * 5`
</ParamField>

<ParamField path="timezone" type="string" required>
  The timezone in which the automation is configured to run in lowercase. You
  can find the list of timezones here: https://timezonedb.com/time-zones
</ParamField>

<ParamField path="team" type="string">
  The identifier of the team to which the automation belongs.
</ParamField>
